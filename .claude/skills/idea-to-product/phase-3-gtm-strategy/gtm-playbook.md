## Channel-Specific Guides

### Reddit Launch Checklist

```markdown
## Reddit Launch Checklist

### Pre-Launch
- [ ] Account has 100+ karma
- [ ] Active in target subreddits for 2+ weeks
- [ ] Read and understand each sub's rules
- [ ] Identified best posting times
- [ ] Prepared authentic story (not sales pitch)
- [ ] Created demo GIF/video

### Post Strategy
- [ ] Title is compelling but not clickbait
- [ ] Post tells a story (problem â†’ solution)
- [ ] Includes visual demo
- [ ] Acknowledges limitations honestly
- [ ] Has clear but soft CTA
- [ ] Offers community-specific discount

### Post-Launch
- [ ] Respond to every comment within 1 hour
- [ ] Thank critics, address concerns
- [ ] Don't get defensive
- [ ] Share genuine updates/learnings
- [ ] Follow up with interested users via DM
```

### Product Hunt Launch Checklist

```markdown
## Product Hunt Launch Checklist

### 2 Weeks Before
- [ ] Create PH account if new
- [ ] Build email list of 50-100 supporters
- [ ] Prepare all assets (logo, screenshots, GIF)
- [ ] Write and refine description
- [ ] Draft maker's first comment
- [ ] Reach out to hunter (optional)

### 1 Week Before
- [ ] Test all links
- [ ] Prepare email to supporters
- [ ] Schedule social posts
- [ ] Brief team on response protocol
- [ ] Prepare FAQ document

### Launch Day
- [ ] Launch at 12:01am PST
- [ ] Send email to supporters immediately
- [ ] Post on social channels
- [ ] Monitor and respond to all comments
- [ ] Share updates every 2-3 hours
- [ ] Thank supporters publicly

### Post-Launch
- [ ] Send thank you email to voters
- [ ] Publish "lessons learned" content
- [ ] Follow up with high-intent comments
- [ ] Update product based on feedback
```
